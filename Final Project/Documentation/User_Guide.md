## User Guide

## Features
- Create tickets
- View tickets
- Track statuses
- Add internal comments
- Assign technicians/Helpdesk Agent
- View SLA deadlines

## How to Use
1. Open the UI HTML page
2. Submit a ticket form
3. View tickets in the list page
4. Backend logs will show operations
5. Oracle tables will update accordingly

## API Endpoints
GET /companies
POST /companies
GET /tickets
POST /tickets

## Troubleshooting
- Ensure Oracle connection is correct
- Check FastAPI logs for errors
